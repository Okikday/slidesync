# SlideSync *(Work in Progress)*

A cross-platform Flutter app for students to organize, access, and sync their educational materials in form of PDFs, images, and documents with smart file management, reading tracking, and real-time WiFi syncing.

---

## Features

- **Module organisation** — Group materials into modules and sub-groups. Categorize by type: questions, materials, and more.
- **PDF & document viewer** — Read PDFs and images in-app with full-screen support and efficient memory management.
- **Reading tracker** — Tracks recently accessed files with a "continue reading" prompt when you return to the app.
- **WiFi sync & casting** — Sync files across devices and cast to a desktop on the same network for real-time viewing. (Work in progress)
- **Smart file management** — Batch rename, reorder, auto-sort, copy, move, and extract files with intelligent naming support.
- **Cross-platform** — Built with Flutter. Runs on Android and Windows.
